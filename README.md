#Начало
Что такое django и как начать его использовать
Что такое Django? Django - это фраeймворк который используется в Python для создание сайтов совместно JS, HTML, CSS.
Для того чтобы его установить напишите pip install django или же если у вас Mac pip3 install django или же Ubuntu создаете виртуальное окружение после чего классическим способом скачиваете его далее вам надо его запустить для этого прописываете в терминале django-admin startproject <<Название папки>>. И начинайте погружатся в мир Django!

#Значение-Папок-Django:
init- написаны характеристики проекта.
asgI - Обеспечивает корректное подключение к серверу.

- Settings :
SECRET-KEY-Описывает все глобальные характеристики (Никогда не делитесь Секретным ключом!, DEBUG- будет показывать ошибки на сайте напрямую, при выгружении или же деплоем ставить False так как П будет лишним знать конкретные подробные ошибки. Ну и вы же не хотите получить от тимлида и аналитика по шапке верно ведь?
ALLOWED_HOSTS нужен для того чтобы в [] вставлять домен по которому сможет П найти ваш сайт.
INSTALLED_APPS - Описывает что уже встроено заранее для работы с Django по типу Admin, messages и подобные.
MIDDLEWARE - Записаны различные промежуточные ПО которые обеспечивают безопасность, Классы и подобные.
ROOT-URLCONF- Основной URL который будет использоваться для проекта <<Имя папки>>.
LANGUAGE__CODE - для смены языка.
TIME_ZONE - Сменить временную зону.
